# Walkthrough — Prompts 9 & 10: Dispatch + Report Generator

## Prompt 9: Dispatch Module ([tools/dispatch.py](file:///c:/Users/bvspr/.gemini/Projects/Food to Go/tools/dispatch.py))

### Architecture

Pure deterministic Python — zero LLM calls. Accepts **async callable parameters** instead of McpToolset objects for full testability.

**Boundary**: Runs after orchestrator Phase 2 (single_store_candidate, StockLedger deduction, Order creation). Does not call any of those.

### 5-Step Fallback Chain (per batch of ≤2 orders, per store)

| Step | Mechanism | Constraint |
|------|-----------|------------|
| 1 | Nearest volunteer | capacity_kg ≥ payload_kg |
| 2 | Next volunteer | total route ≤ 120 min |
| 3 | 15-min detour bundle | `check_detour_bundle()` exposed publicly |
| 4 | Store's own truck | `get_truck_avail` callable |
| 5 | Commercial pickup | Always succeeds; uses `COMMERCIAL_CAPACITY_SENTINEL` |

Each candidate is validated via `DispatchOutput` guardrail — failure falls to next tier.

### New Model Added

`DispatchStats` in [models.py](file:///c:/Users/bvspr/.gemini/Projects/Food to Go/tools/models.py): tracks total/volunteer/truck/commercial counts, unavailable volunteers, urgent fallbacks, bundled detours.

## Prompt 10: Report Generator ([reports/report_generator.py](file:///c:/Users/bvspr/.gemini/Projects/Food to Go/reports/report_generator.py))

### Four Sections

| Function | Output | Notes |
|----------|--------|-------|
| `generate_delivery_table()` | pandas-formatted console table | Urgent items marked ★ |
| `generate_map()` | Folium HTML → `reports/output/map_{run_id}.html` | Green/blue/orange polylines + legend |
| `generate_negotiation_report()` | Text transcript per care home | All 5 homes; auto-accept / rejection noted |
| `generate_audit_report()` | Store inventory + dispatch stats | Pushed vs held-back with threshold context |

`generate_full_report()` calls all four and returns:
```python
{
    "delivery_table": str,
    "map_filepath": str,
    "negotiation_report": str,
    "audit_report": str,
    "message_log": List[dict],
}
```

### Map Details
- Center: Chennai (13.0827, 80.2707), tiles: CartoDB positron
- Store markers → blue FA `shopping-cart`
- Care home markers → green FA `home`
- Volunteer dots → grey `CircleMarker`
- Routes: green (volunteer), blue (store truck), orange dashed (commercial)
- Legend injected as HTML element
